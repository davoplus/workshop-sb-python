# workshop-sb-python
Repositorio para el workshop de seguros bolivar

## Instalacion

1) Si no estas en cloud9 debes instalar python (3.7 o superior)
2) pip install -U pip
3) pip install mysql-connector-python
4) pip install boto3



1) En la linea de comando escribir `git clone https://github.com/davoplus/workshop-sb-python.git`
3) Ahora en la linea de comando escribir `python main.py <secret manager name>`
